# AI Chatbot in Python

This is a simple AI-powered chatbot built with Python and the OpenAI API.  
It allows users to interact with the bot via the terminal and logs all conversations into a CSV file.

---

## Features
- Interactive chatbot using OpenAI GPT model
- Logs conversations with timestamps into `chat_log.csv`
- Easy to run locally with minimal setup

---

## Requirements
- Python 3.10+
- Packages listed in `requirements.txt`
- An OpenAI API key

---
